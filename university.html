Final Year Project

For my final year project at university, I researched into the area of Android Anti-Theft and developed an application for the platform using Mobile Java in Android Studio. The aim of the application was to decrease the theft in smart phones/tablets by offering security functionality should the user misplace their device. This application also featured an external web resource in the form of a remote control in which I created using JavaScript.

The coding of this project involved heavy usage of API’s to allow the communication between the device and remote control. During this project I also acquired skills in project management, product design and automated testing in Android Studio.

I was delighted to receive the offer to allow me to present my final product at the Technology Showcase at Leeds Beckett University. I also received the BCs West Yorkshire branch prize for the best individual project at my university. My project can be found on my GitHub account among other things I have worked/practiced with. https://github.com/andrewhall2469
